# Contributing to Agent 200

First off, thanks for taking the time to contribute!

This project is a submission for the AI Dev Days Hackathon 2026. While we are currently in "Hackathon Mode" (rapid prototyping), we welcome ideas, bug reports, and pull requests from the community.

## How to Contribute

### Reporting Bugs
If you find a bug, please create a new issue and include:
- A clear description of the bug.
- Steps to reproduce.
- Your environment (OS, .NET version).

### Suggesting Enhancements
Have an idea for a new agent skill or MCP integration?
- Open a new issue with the enhancement label.
- Explain why this feature would be useful.

### Submitting Code
1. Fork the repository and create your branch from main.
2. Setup your environment:
   - Install .NET 9.0 SDK.
   - Run `dotnet restore` to install dependencies.
3. Make sure your code builds: Run `dotnet build`.
4. Test your changes: Run `dotnet test`.
5. Submit a Pull Request:
   - Ensure your PR description clearly describes the problem and solution.
   - Link to any relevant issues.

## Development Workflow
This project follows a Feature Branch workflow.
- **main branch**: Contains the latest stable, deployable code.
- **feature/* branches**: Used for new features (e.g., `feature/add-github-mcp`).
